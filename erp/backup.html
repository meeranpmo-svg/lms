Backup Schedule

Recommended Auto-Backup Plan

Daily Backup — 6:00 PM Full ERP data snapshot every day after school hours
Weekly Archive — Every Friday 8:00 PM Compressed weekly archive kept for 4 weeks
Monthly Snapshot — 1st of every month Long-term monthly backup stored for 12 months

Tip: For Docker deployments, use the backup.sh script in the repo root to schedule automatic backups via cron. Run crontab -e and add the cron line from the script comments.
