🔗 Connect Google Drive for Auto-Sync

Paste your Google Cloud OAuth2 Client ID below to enable automatic sync from the Meet Recordings folder.

📋 Get Client ID

In Google Cloud Console → Create OAuth2 Client ID → Web Application → Add your site URL to "Authorized JavaScript Origins".

💡 How to add a Google Drive recording:
1. Open Google Drive → find the recording video   2. Right-click → Share → "Anyone with the link" → Copy link   3. Paste that link below — the LMS converts it to an embed automatically ✅

Add Recording
